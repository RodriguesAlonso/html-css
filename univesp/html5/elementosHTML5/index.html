======================================================>Elemento section

reprsenta uma seção genérica de uma página ou aplicação, um agrupamento de conteúdo.
Não é um elemento para gerar layout como o elemento div
exemplo:

Avaliação de interfaces

Uma proposta de processo de avaliação

O sucesso ou fracasso de um site está diretamente relacionado ao design de sua interface. Seguindo esta premissa a interface de um website deve atender a requisitos como os de usabilidade, comunicabilidade, navegabilidade e acessibilidade.

Introdução

A interface de usuário é a parte de um sistema computacional com a qual uma pessoa entra em contrato de forma física, perceptiva ou ainda conceitual funcionando como o meio de interação com a aplicação.

Objetivo

O objetivo desse projeto é elaborar uma proposta de processo de avaliação da interface de websites institucionais que contemple os critérios de usabilidade, comunicabilidade, navegabilidade e acessibilidade.

======================================================>ELEMENTO ASIDE

seção de uma página para conteúdo periférico ao conteúdo do documento preincipal
exemplo

A UML pe uma linguagem oficializada no final de 1997 que surgiu da iniciativa de um grupo de pesquisadores que, a partir da investigação de métodos reconhecidos mundialmente como os principais métodos orientados a objetos, definiram uma gramática e um conjunto de regras sintáticas para modelar projetos de softwares orientados a objeto.

A UML é adequada para a modelagem de sistemas, cuja abrangência poderá incluir sistemas de informação corporativos(Booch et, al, 2009).

Apesar de a UML ser especialmente utilizada no desenvolvimento de softwares orientados a objetos, ela é suficientemente expressiva para modelar sistemas que não sejam de software, como por exemplo, projetos de hardware.

======================================================>ELEMENTO FIGURE E FIGCAPTION

O elemento figure representa um conteúdo de fluxo autossuficiente e opcionalmente com legenda, como imagens, diagrams, fotos, código.

O elemento Figcaption é o primeiro elemento filho de figure, representa a legenda do coteudo figure

exemplo

Um dos melhores jogos no mundo é o Pokemom, sua primeira versão contia como pokemons iniciais o squartle, bulbassauro e charmander

Olhando para os pokemons o Blastoise é o melhor.
